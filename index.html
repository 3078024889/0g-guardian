⚙️ How 0G Guardian Works
1
Request Submitted On-Chain
User submits contract address to AuditRegistry.sol on 0G Chain. Smart contract emits AuditRequested event.
0G Chain · EVM · Solidity 0.8.20
2
Static Analysis — Slither Engine
Guardian Agent detects the event, fetches source code, runs Slither static analysis. Identifies reentrancy, access control flaws, integer overflow, unsafe delegatecall patterns.
Slither · Python · Pattern Matching
3
AI Semantic Analysis — 0G Compute
Source code + static findings sent to LLM via 0G Compute Network. AI reasons about business logic flaws, economic attacks, governance manipulation — things static tools miss.
0G Compute · Qwen2.5-72B · OpenAI-compatible API
4
Report Stored on 0G Storage
Full JSON audit report (with all findings, attack vectors, recommendations) uploaded to 0G Storage. Returns content-addressed merkle hash for integrity verification.
0G Storage · Content-Addressed · Permanent
5
Risk Score Written to 0G Chain
AuditRecord written on-chain: risk level, numeric score (0–100), storage hash, severity counts. Immutable forever. Any dApp can call isContractSafe(address).
0G Chain · AuditRegistry.sol · Composable
